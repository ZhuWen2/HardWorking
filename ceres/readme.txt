1.使用方法
1）将include文件夹加入到工程的包含目录
2）将lib文件夹加入的工程的库目录
3）将lib文件夹中的导入库加入到工程中的链接输入列表里(目录下有release.txt和debug.txt可以直接复制对应的文件名)
4）将bin目录加入到环境变量的Path中。（或将bin下的dll复制到可执行文件目录）

2. 可能遇到的问题
1）C3861“M_2_SQRTPI”: 找不到标识符		
ceres使用了非C++标准的宏定义,而vs默认是禁用非C++标准定义，解决办法：
a.在工程的预处理器定义里加入 "_USE_MATH_DEFINES" 启用非标准宏定义。
或则和 b.关闭标准符合性或者符合模式选项，即设置为否(c/C++  ->  语言  ->  符合模式)
